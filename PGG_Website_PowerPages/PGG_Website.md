# Power Gym Girls Project - Power Pages Website

Our group also created a website using Power Pages, as a way to advertise the gym's branding, especially available fitness classes and some members' inspiring stories. While browsing the website, the user also has access to Mojo Jojo, our assistant/chatbot.
